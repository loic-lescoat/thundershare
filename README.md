# Usage

## Requirements

[`make`](https://www.gnu.org/software/make/) and [`docker`](https://docs.docker.com/engine/install/)

## Installation instructions

Clone this repo, then run:

```
# build app
make build

# run app
make run

# access the app at <machine ip>:8000
```
